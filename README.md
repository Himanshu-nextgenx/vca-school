# Virat Children Academy — School Website

A fully responsive, modern, premium-level school website built with **React.js (Vite) + TailwindCSS**.

---

## 🏫 School Details
- **School**: Virat Children Academy (VCA)
- **Location**: Narayanpura, Railway Station Road, Kuchaman City
- **Session**: 2026–27
- **Boards**: CBSE & RBSE
- **Classes**: Nursery to 12th (Arts & Science)

---

## 🚀 Getting Started

### Prerequisites
- Node.js v18+ 
- npm v9+

### Installation
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will be available at: **http://localhost:5173**
